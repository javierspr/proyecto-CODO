Elementos en bloque y en línea

Elementos en bloque (block)

Los elementos en bloque son aquellos que crean un "bloque" dentro del flujo normal del contenido y ocupan todo el ancho disponible del contenedor principal. Esto significa que comienzan en una nueva línea y empujan a otros elementos a ocupar líneas separadas.
Algunos ejemplos de elementos en bloque son < div >, < p >, < h1 > a < h6 >, < ul >, < ol >, < li >, < table >, < form >, entre otros.
Estos elementos se utilizan para estructurar y dividir el contenido de una página en secciones.

El elemento < div > se utiliza principalmente como un contenedor genérico y versátil para agrupar y estructurar otros elementos dentro de una página web.

Las Tablas y Formularios también son elementos en bloque.

Elementos en línea (inline)

Los elementos en línea son aquellos que se ajustan dentro del flujo normal del contenido y solo ocupan el espacio horizontal necesario para mostrar el contenido. No comienzan en una nueva línea y permiten que otros elementos en línea compartan la misma línea.
Algunos ejemplos de elementos en línea son < span >, < a >, < strong >, < em >, < img >, < br >, < input >, < button > , entre otros.
Estos elementos se utilizan para aplicar estilo o interactividad a partes específicas del contenido.

Ejemplo de elementos en línea:

* El elemento < span > es utilizado para aplicar estilos o realizar manipulaciones específicas en partes individuales de un texto o contenido en línea, sin cambiar el significado semántico del contenido.

Es importante comprender la diferencia entre estos dos tipos de elementos, ya que afecta la presentación y el diseño de una página web. Los elementos en bloque son útiles para crear la estructura general de la página, mientras que los elementos en línea son ideales para aplicar estilos y funcionalidades a partes individuales del contenido.
